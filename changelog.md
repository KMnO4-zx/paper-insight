# Paper Insight 每天都在变好，希望大家多多反馈、支持！

## 2026-07-14

- “我的论文”和论文合集页新增阅读概览：通过近 16 周日历热力图展示每天首次看过的论文数量，并支持悬停查看具体日期和篇数。
- 新增连续阅读、本月阅读、今日阅读统计，以及 Hugging Face Daily Papers 最新榜单 Top 5 阅读进度。
- “我的论文”会展示全部论文合集进度，进入具体合集后只展示当前合集进度；桌面端支持悬浮跟随，移动端完成自适应布局。

## 2026-07-06

- 新增 ACL 2026 论文合集，并加入后台关键词补全能力，方便持续完善论文检索和标签信息。
- 新增 ICML 2026 OpenReview 元数据下载工具，为后续论文合集导入做准备。

## 2026-07-04

- 论文正文读取改用 Jina Reader 的 POST JSON 接口，减少匿名 GET 请求被拦截导致的读取失败，并继续避免缓存访问验证页。
- Hugging Face Daily Papers 详情页关键词支持点击跳转到 HF Daily 合集内的关键词搜索结果。
- 修复论文详情页对话面板在不同屏幕高度下的尺寸和输入区域定位问题。

## 2026-06-28

- 修复代码开源状态筛选上线后导致的冷搜索变慢问题，恢复 PostgreSQL 全文搜索索引下推。

## 2026-06-27

- 论文新增代码开源状态标签，详情页和论文列表页都会展示是否有公开代码 🧑‍💻
- 论文合集、搜索、HF Daily 和 arXiv 页面支持按代码开源状态筛选，方便快速找到可复现论文。
- 管理员后台新增代码开源状态判断任务，并将后台任务区改为层叠卡片切换，支持保留上次查看的任务卡片 🃏
- “Open in AI” 的提示词改为后端统一管理，后续调整讲解提示词更方便 ✨

## 2026-06-25

- 优化论文搜索：改为标题、摘要、关键词三路候选召回，并按字段命中权重重新排序，常见关键词和无结果搜索更快。
- Docker 构建改用 uv 和 lockfile 安装 Python 依赖，减少与本地开发环境的依赖差异。
- 后台用户管理新增注册时间、最近登录和在线状态排序，并支持刷新当前在线人数。
- 大模型配置的测试结果改为显示在配置卡片内，减少页面顶部提示对其他模块的干扰。

## 2026-06-24

## 2026-06-23
